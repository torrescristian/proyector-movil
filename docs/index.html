# 📽 Proyector Móvil

# ¿Qué es Proyector Móvil?

Proyector Móvil es una aplicación web y de escritorio que permite emitir y mostrar imágenes (filminas) a dispositivos remotos conectados a una misma red local.

# Descargas

  • Windows 32 / 64 bits
  • Linux
